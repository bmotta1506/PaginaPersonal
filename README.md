Nombre del Proyecto: Libreria Online Entre Hojas

URL del proyecto hospedado: [http://bmottita.pythonanywhere.com](http://bmottita.pythonanywhere.com)

Descripción
Entre Hojas es una plataforma web diseñada para amantes de la lectura, que permite a los usuarios registrarse, buscar libros y comprar a través de una interfaz amigable e intuitiva.
El sistema gestiona una base de datos de usuarios para registrarse e iniciar sesion, ofrecer una experiencia organizada y eficiente.

Lenguajes utilizados
HTML5 y CSS3 para el diseño y estructura del Frontend.
JavaScript para validaciones e interacciones básicas.
Python (Flask) para la lógica del servidor y rutas.
SQL (MySQL) para la gestión y persistencia de datos.
